## 🛠️ Installation

Clone the repo:

```bash
git clone https://github.com/yourusername/creatiAI.git
```

Install NPM packages:

```bash
npm install
```

Set up environment variables:

```bash
cp .env.example .env.local
```

Add your configuration details to `.env.local`.

Start the development server:

```bash
npm run dev
```
